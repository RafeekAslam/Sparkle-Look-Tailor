# Sparkle Look Tailor Website ✨

A stylish and responsive tailor shop website built using HTML and CSS.

## 💡 Features
- Elegant and modern design
- Fully responsive layout
- Service showcase section
- Image gallery for designs
- Contact section

## 🛠️ Technologies Used
- HTML5
- CSS3
- Javascipt (basic)
